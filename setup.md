# Setup Instructions

## Clone Repository

```bash
git clone https://github.com/username/CIFAR10-Image-Classification.git
```

## Install Dependencies

```bash
pip install -r requirements.txt
```

## Run Notebook

Open

```
notebooks/cifar-10-image-classification.ipynb
```

using

- Kaggle
- Google Colab
- Jupyter Notebook

TensorFlow automatically downloads the CIFAR-10 dataset during execution.
