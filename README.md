# CIFAR-10 Image Classification using CNN

## Overview
This project implements an image classification model using a Convolutional Neural Network (CNN) on the CIFAR-10 dataset. The trained model classifies images into one of ten categories.

## Dataset
- CIFAR-10
- 60,000 images
- 10 classes
- Image size: 32×32 RGB

## Classes
- Airplane
- Automobile
- Bird
- Cat
- Deer
- Dog
- Frog
- Horse
- Ship
- Truck

## Model
- CNN
- Batch Normalization
- Dropout
- Data Augmentation
- Adam Optimizer
- Early Stopping

## Technologies
- TensorFlow / Keras
- NumPy
- Matplotlib
- OpenCV
- Scikit-learn
